📖 Linear & Polynomial Regression
Linear: y = β₀ + β₁t
Polynomial: y = β₀ + β₁t + β₂t²

Dimana:
• y = volume distribusi
• t = indeks waktu (0, 1, 2, ...)
• β₀ = intercept (nilai awal)
• β₁ = slope (laju perubahan)

Kelebihan: mudah diinterpretasi, baseline yang baik
Kekurangan: asumsi linearitas, sensitif terhadap outlier

Evaluasi: MAE, RMSE, R² Score
📖 Moving Average & Exponential Smoothing
Moving Average: MA(t) = Σ y(t-i) / w
Exp Smoothing: S(t) = αy(t) + (1-α)S(t-1)

Dimana:
• w = window size (default: 3 bulan)
• α = smoothing factor (0 < α < 1)
• α besar → lebih responsif terhadap data baru
• α kecil → lebih stabil, kurang reaktif

Kelebihan: meredam noise, adaptif
Kekurangan: lag terhadap perubahan mendadak

Default α: 0.3 (konservatif)
